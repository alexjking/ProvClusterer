About

ProvClusterer is an interactive tool to create a simplified view of a provenance document.

This tool identifies clusters of elements within a document using the Girvan-Newman clustering algorithm. Each group of elements can be collapsed into a single element resulting in a smaller and less complex document.

This was created by Alexander King as part of a third year project for MEng Computer Science at the University of Southampton and is built using the ProvToolbox which provides an implementation of the PROV specification.
